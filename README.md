# 📊 Naver ETF Real-time EDA Dashboard

네이버 금융 실시간 ETF API 데이터를 기반으로 다차원 탐색적 데이터 분석(EDA)을 제공하며, **GitHub Pages**를 통해 무상태(Stateless) 및 서버리스로 즉시 호스팅할 수 있도록 정밀 설계된 반응형 웹 대시보드 애플리케이션입니다.

---

## 🛠️ 핵심 기술 스택 및 아키텍처
* **Frontend**: Vanilla HTML5, JavaScript (ES6+), [Tailwind CSS (CDN)](https://tailwindcss.com), [Lucide Icons](https://lucide.dev)
* **Visualization**: [Apache ECharts (CDN)](https://echarts.apache.org) - 11종의 정밀 통계 및 금융 인터랙티브 그래프
* **Data Processing**: Python 3.11+, Pandas, NumPy (백업 데이터 갱신용 엔진)
* **Automation (CI/CD)**: GitHub Actions (5분 주기 자동 갱신 및 웹 빌드/배포)

---

## 💡 주요 아키텍처 특징

### 1. CORS 우회 및 하이브리드 데이터 로드
* **실시간 데이터 수집 (JSONP)**: 브라우저 단에서 직접 네이버 API(`https://finance.naver.com`)에 요청을 송신할 때 발생하는 보안 제약(CORS)을 우회하기 위해 **JSONP 방식**(`<script>` 동적 주입 및 콜백 호출)을 탑재하여 완벽한 실시간 데이터 분석을 지원합니다.
* **Fallback 세이프가드**: 브라우저 환경이나 네트워크 차단으로 실시간 API 로드가 지연될 경우, 저장소 내에 적재된 최신 백업 데이터셋(`data/etf_fallback.json`)을 즉시 자동으로 Fetch하여 대시보드가 상시 무중단으로 가동되도록 우회 제어합니다.

### 2. 고도의 웹 성능 최적화 (Speed Optimization)
* **즉시 로딩 (0초 진입)**: 페이지 접속 시 대기 시간이 소요되는 실시간 API 대신, 물리적으로 압축된 로컬 백업 JSON을 우선적으로 즉시 로딩하여 첫 화면 전환 속도를 극대화했습니다.
* **지연 렌더링 (Lazy Tab Rendering)**: 비활성화 상태의 탭에 속한 대용량 ECharts 시각화 노드 및 요약 테이블을 최초 진입 시 한꺼번에 그리지 않고, 사용자가 탭을 전환하여 활성화하는 시점에만 지연 로딩(Lazy Initialization)하도록 설계하여 브라우저 연산 병목을 제거했습니다.
* **JSON 경량화 (Minification)**: 스크립트가 내보내는 JSON 파일에서 모든 공백과 줄바꿈을 제거한 Compact JSON 포맷을 사용하여 네트워크 전송 비용을 최소화했습니다.

## 📊 대시보드 주요 기능 및 탭 구성

### 1. 대시보드 개요 (Overview)
* **주요 핵심 지표**: 총 상장 ETF 수, 시장 전체 시가총액 합계, 당일 총 거래대금, 실시간 동기화 출처 배지 표출.
* **기술 통계 요약**: 수치형 변수(현재가, 시가총액, 등락률, 수익률, 거래량, 거래대금)에 대하여 표본 통계 공식을 자체 연산하여 **평균, 중앙값, 표준편차, 왜도(Skewness), 첨도(Kurtosis)** 요약 표를 제공합니다.
* **운용사 점유율**: 자산운용사별 상장 수, AUM 비중 정보 테이블 및 ECharts 기반 가로 막대 그래프 시각화.

### 2. 인터랙티브 EDA (Charts - 11종)
각 차트 하단에는 **원천 통계 데이터 요약 표**와 **300자 이상의 정교한 한국어 분석 인사이트 및 액션 플랜**이 내장되어 있습니다.
1. **시가총액 상위 Top 10** (개별 ETF 규모 순위)
2. **거래대금 상위 Top 10** (당일 유동성 쏠림 추적)
3. **3개월 수익률 구간별 분포** (단기 수익률 빈도 히스토그램)
4. **운용사별 ETF 상장 종목 수** (브랜드별 양적 라인업 비교 막대 차트)
5. **운용사별 시가총액 합계 점유율** (브랜드별 AUM 시장 지배력 비교 막대 차트)
6. **거래량 vs 시가총액 상관 분석** (활동성-규모 간 분산 산점도)
7. **3개월 수익률 vs 거래대금 상관 분석** (추세 추종 모멘텀 검증 산점도)
8. **일일 등락률 구간별 분포** (시장 전체 탐욕-공포 센티먼트 히스토그램)
9. **NAV 괴리율 분포** (순자산가치 대비 시장 왜곡 오차 검증)
10. **운용사별 평균 3개월 수익률 비교** (운용 트렌드 적합성 비교 막대 차트)
11. **운용사 및 종목별 시가총액 트리맵 (Treemap)**: 운용사별 시총 지분 기하학적 면적 시각화 및 상자 클릭 시 하위 종목별 시총 규모가 나타나는 **실시간 드릴다운(Drill-down)** 지원.

### 3. 종목 탐색기 (Explorer)
* 종목명 및 종목코드 실시간 검색 와일드카드 지원.
* 자산운용사 브랜드 필터 및 시가총액/거래대금/괴리율/수익률 기준 실시간 오름차순/내림차순 정렬 조회 테이블.

### 4. 무결성 검증 (Data Validation)
* **모순성 검증 (Logical Consistency)**: 현재가/NAV 음수 이탈, 거래량 존재 시 거래대금 누락, NAV 괴리율 임계치(3%) 극단 이탈 등 비정상 데이터 항목 자동 감시.
* **수학적 교차 검증 (Cross Validation)**: 개별 운용사별 시가총액 집합 합산 값과 전체 ETF 원천 데이터 합계량의 일치 여부를 소수점 단위까지 검증 판정.

## 🤖 GitHub Actions 배포 자동화 파이프라인
* **자동 크론 수집**: 매 5분 주기(cron: `*/5 * * * *`)로 가상 환경 내에서 Python 수집기(`src/collect_fallback.py`)를 실행하여 최신 데이터를 `data/etf_fallback.json` 파일에 저장합니다.
* **커밋 및 배포**: 데이터 갱신 시 `[skip ci]` 태그를 포함하여 리포지토리에 자동 푸시하고, 최신 정적 웹 페이지를 GitHub Pages 호스팅 서비스로 완전 자동 재빌드 및 재배포합니다.
